# Print-It

## Description
Développement d'un carrousel interactif en hero pour un site d'imprimerie : navigation par flèches, réalisée en JavaScript. Le reste du site était déjà fourni.

Development of an interactive hero carousel for a printing company website: arrow-based navigation, built in JavaScript. The rest of the site was already provided.

## Deployment
*Ce projet est déployé via GitHub Pages.* 
printIt repo > Settings > Pages > Sélectionner la branche > Save

*This project has been deployed on Github Pages*
printIt repo > Settings > Pages > Select branch > Save
